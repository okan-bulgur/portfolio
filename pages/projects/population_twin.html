Population Twin

Earthquakes are a serious reality — especially for countries like ours where risk is high. This project aims to make that reality easier to manage using digital technology.


We developed a web-based digital twin that helps visualize and analyze key information about buildings directly on a map. Things like building type, age, number of floors, fire escape availability, and population density can be added and instantly visualized.



What it does:


  • Maps out buildings with detailed info
  • Calculates earthquake vulnerability based on building data
  • Updates visuals dynamically to reflect each building’s risk level
  • Exports data in standard XML format for use in other systems (e.g. decision support tools)


Why it matters:


By turning physical data into a digital, interactive format, this app helps city planners, emergency responders, and local authorities make faster and smarter decisions. It also creates a space where everyone — from data collectors to policy makers — can work together more efficiently.



Goal: To help make cities more earthquake ready through better communication, smart data, and visual decision tools.
